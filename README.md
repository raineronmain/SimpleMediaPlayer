# SimpleMediaPlayer
A simple media player, made with kotlin + javafx (tornadofx)

Supports Drags & Drop

Controls:

- Play / Pause    -   LeftMouse click / Space

Skip:

- Skip Fwd        -   1s  ( -> ) | 5s  crtl + ( -> )
- Skip Rev        -   1s  ( <- ) | 5s  crtl + ( <- )
- Skip Fwd/Rev    -   MouseWheel

Zoom In / out
-   crtl + MouseWheel
- Reset Zoom & pan-   click MouseWheel
- 
*Tested with Bellsoft LibericaJVM 21 LTS (full = JVM + JavaFx).*

*on:*
- *Linux Mint 22.1*
- *Windows 7 + 10*
